7  假设检验

7.0.1 第一类错误

  • 拒真概率： 当原假设为真时，检验统计量落入拒绝域的概率 \[P(X \in W |H_0=TRUE)\]

  • \(\alpha\)

7.0.2 第二类错误

  • 取伪概率：当备择假设为真时，检验统计量未落入拒绝域的概率

\[P(X \notin W |H_1=TRUE)\]

  • \(\beta\)

  • 假验的效能：正确接受备择假设的概率 \[(1-\beta) \times 100\%\]

7.0.3 \(p\)值

7.0.4 样本量

  • 效能分析：决定统计学检验的效能和计算揭示一定大小效应所需要的最小样本量
    • \(\alpha\): 第一类错误概率
    • \(\beta\): 第二类错误概率
    • \(d\): 效应的大小，即所研究的效应相对样本的标准差\(\sigma\)的大小
    • \(n\): 样本大小
  • 上面四个参数，只要固定三个，就可以确定剩余参数的值

7.1 题目

7.1.1 灯泡寿命是否为理论值

某型灯泡寿命X服从正态分布,从一批灯泡中任意取出10只,测得其寿命分别为(单位:h) \[1490, 1440, 1680, 1610,1500, 1750, 1550, 1420, 1800, 1580\] 能否认为这批灯泡平均寿命为1600h（\(\alpha=0.05\)）

a = c(1490, 1440, 1680, 1610,1500, 1750, 1550, 1420, 1800, 1580)
xbar = mean(a)
n = length(a)
tvalue = (xbar - 1600) / (sd(a)/sqrt(n))
tvalue
[1] -0.442742
alpha = 0.05
tpoint = qt(1-alpha/2, n-1)
tpoint
[1] 2.262157
  • tvalue未落入拒绝域，接受原假设

7.1.2 改进工艺后方差是否为理论值

某炼钢厂铁水含碳质量分数X在正常情况下服从正态分布.现对操作工艺进行了改 革又测量了5炉铁水,含碳质量分数分别为： \[4.421, 4.052, 4.357, 4.287, 4.683\] 是否可以认为由新工艺炼出的铁水含碳质量分 数的方差仍为\(0.108^2\)

a = c(4.421, 4.052, 4.357, 4.287, 4.68)
n = length(a)
chiValue = ((n-1) * var(a)) / (0.108^2)
chiValue
[1] 17.68872
alpha = 0.05
chiLeftPoint = qchisq(alpha/2, n-1)
chiRightPoint = qchisq(1-alpha/2, n-1)
chiLeftPoint
[1] 0.4844186
chiRightPoint
[1] 11.14329
  • chiValue落入拒绝域，拒绝原假设

7.1.3 药物药效分析

某种物种在处理前与处理后取样分析其含脂率如下: 处理前: \[0.19, 0.18, 0.21, 0.30, 0.66 0.42, 0.08, 0.12, 0.30 , 0.27\] 处理后: \[0.15, 0.13, 0.00, 0.07, 0.24, 0.19, 0.04, 0.08, 0.20, 0.12\] 假定处理前后含脂率都服从正态分布, 且相互独立, 方差相等. 问处理前后含脂率的均值有无显著差异（\(\alpha=0.05\)）

  • 建立原假设与备择假设 \[ H_0: \mu_1 = \mu_2, H_1: \mu_1 \neq \mu_2 \]

  • 检验统计量 \[ T = \frac{\bar{X}-\bar{Y}}{s_w^* \sqrt{\frac{1}{n_1}+\frac{1}{n_2}}}\]

  • 拒绝域

\[D =\left\{w \middle| |Z| > \mu_{\alpha/2}\right\}\]

a = c(0.19, 0.18, 0.21, 0.30, 0.66, 0.42, 0.08, 0.12, 0.30 , 0.27)
b = c(0.15, 0.13, 0.00, 0.07, 0.24, 0.19, 0.04, 0.08, 0.20, 0.12)
n1 = length(a)
n2 = length(b)
xbar = mean(a)
ybar = mean(b)
sw = sqrt(((n1-1)*var(a) + (n2-1)*var(b))/(n1+n2-2))
# varx = sum((a - mean(a))^2)/(n1-1)
# vary = sum((b - mean(b))^2)/(n2-1)
# sw = sqrt(((n1-1)*varx + (n2-1)*vary)/(n1+n2-2))
tvalue = (xbar-ybar)/(sw * sqrt(1/n1+1/n2))
tvalue
[1] 2.595701
alpha = 0.05
tAlpha = qt(1-alpha/2, n1+n2-2)
tAlpha
[1] 2.100922
  • tvalue落入拒绝域，拒绝原假设，均值有显著差异

7.1.4 断裂强度方差

为了考察温度对某物体断裂强度的影响,在 70度 与 80度下分别重复作了8次试验,得断裂强度的数据如下(单位：Mpa):

70度: \[20.5, 18.8, 19.8, 20.9, 21.5, 19.5, 21.0, 21.2\] 80度: \[17.7, 20.3, 20.0, 18.8, 19.0, 20.1, 20.2, 19.1\]

假设断裂强度服从正态分布, 试问两种情形下断裂强度方差是否相同（\(\alpha=0.05\)）

a = c(20.5, 18.8, 19.8, 20.9, 21.5, 19.5, 21.0, 21.2)
b = c(17.7, 20.3, 20.0, 18.8, 19.0, 20.1, 20.2, 19.1)
varX = var(a)
varY = var(b)
Fvalue = varX/varY
Fvalue
[1] 1.068966
n1 = length(a)
n2 = length(b)
alpha = 0.05
Fleft = qf(alpha/2,n1-1, n2-1)
Fright = qf(1-alpha/2,n1-1, n2-1)
Fleft
[1] 0.2002038
Fright
[1] 4.994909
  • Fvaleu未落入拒绝域，接受原假设
