6.3.1

从自动机床加工的同类零件中抽取16件， 测得长度分别为(单位:cm): \[ 12.15, 12.12, 12.01, 12.08, 12.09, 12.16, 12.06, 12.13, 12.07, 12.11, 12.08, 12.01,12.03, 12.01, 12.03, 12.06\] 假设零件长度服从正态分布，求总体长度方差与标准差的95%置信区间.

a = c(12.15, 12.12, 12.01, 12.08, 12.09, 12.16, 12.06, 12.13, 12.07, 12.11, 12.08, 12.01,12.03, 12.01, 12.03, 12.06)
n = length(a)
alpha = 0.05
leftPonit = (n-1)*var(a) / (qchisq(1-alpha/2, n-1))
rightPonit = (n-1)*var(a) / (qchisq(alpha/2, n-1))
leftPonit
[1] 0.001331471
rightPonit
[1] 0.005844649

6.3.2 双正态总体均值差的区间估计(方差已知)

两台机床生产同一个型号的滚珠，从甲机 床生产的滚珠中抽取8个，从乙机床生产的滚珠 中抽取9个，测得这些滚珠的直径（mm）如下: 甲机床：\[15.0, 14.8, 15.2, 15.4, 14.9, 15.1, 15.2, 14.8\]

乙机床：\[15.2, 15.0, 14.8, 15.1, 15.6, 14.8, 15.1, 14.5, 15.0\]

若两台机床生产的滚珠直径的标准差分别是0.18与0.24求这两台机床生产的 滚珠直径均值差置信度为0.90的置信区间.

x = c(15.0, 14.8, 15.2, 15.4, 14.9, 15.1, 15.2, 14.8)
y = c(15.2, 15.0, 14.8, 15.1, 15.6, 14.8, 15.1, 14.5, 15.0)
xbar = mean(x)
ybar = mean(y)
n1 = length(x)
n2 = length(y)
alpha = 0.1
mupoint = qnorm(1 - alpha/2)
leftPoint = (xbar - ybar) - sqrt((0.18^2)/n1 + (0.24^2)/n2) * mupoint
rightPoint = (xbar - ybar) + sqrt((0.18^2)/n1 + (0.24^2)/n2) * mupoint
leftPoint
[1] -0.1292567
rightPoint
[1] 0.2070344

6.3.3 双正态总体均值差的区间估计(方差相等且未知)

机床厂某日从两台机床加工的零件中，分别抽取若干个样品，测得零件的尺寸分别如下（单位：cm）： A台：\[6.2, 5.7, 6.5, 6.0, 6.3, 5.8, 5.7, 6.0, 6.0, 5.8, 6.0 \] B台: \[5.6, 5.9, 5.6, 5.7, 5.8, 6.0, 5.5, 5.7, 5.5 \] 假设两台机器加工的零件尺寸均服从正态分布,
且方差相等, 取置信度为0.95, 试求两台机器 加工的零件平均尺寸之差的区间估计.

x = c(6.2,  5.7,  6.5,  6.0, 6.3,  5.8, 5.7, 6.0,  6.0,  5.8,  6.0 )
y = c(5.6,  5.9,  5.6,  5.7,  5.8,  6.0,  5.5, 5.7,  5.5)
n1 = length(x)
n2 = length(y)
xbar = mean(x)
ybar = mean(y)
sw2 = ((n1-1) * var(x) + (n2 - 1) * var(y))/(n1+n2-2)
sw = sqrt(sw2)
alpha = 0.05
tpoint = qt(1-alpha/2, n1+n2-2)
leftPoint = (xbar - ybar) - tpoint * sw * sqrt(1/n1+1/n2)
rightPot = (xbar - ybar) + tpoint * sw * sqrt(1/n1+1/n2)
leftPoint

rightPoint
